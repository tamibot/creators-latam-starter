---

Invocá el skill `plan-maestro` para generar el scaffold inicial de `documentation/plan-maestro.md`.

Pasos:
1. Preguntame nombre del cliente, nombre del proyecto, fecha inicio (o usar hoy).
2. Confirmá el nombre del responsable del lado Creators Latam.
3. Generá el scaffold con esos datos reemplazando los placeholders.
4. Validá que el archivo se creó correctamente.
5. Sugerime invocar al agente `kickoff-cliente` (si aún no hubo primera reunión) o al `onboarding-pm` (si ya hay brief inicial) para completar las 3 fases.

Si ya existe un `documentation/plan-maestro.md`, pedime confirmación antes de sobreescribir.
