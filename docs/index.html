PREGUNTAS FRECUENTES

Lo que más nos preguntan.

Respuestas cortas. Si no está acá, escribinos.

¿Necesito ser técnico para usar esto?
No. Los 5 pasos están pensados para que cualquiera que clone y siga el onboarding pueda lanzar un proyecto. El sistema pregunta, vos respondés.
¿Cuánto tarda la instalación?
Primera vez en Mac vacía: 15–30 min. Siguientes corridas: segundos (el script detecta qué ya tenés y salta lo instalado).
¿Funciona en Windows?
El script asume macOS. En Windows pedile a Claude/Cursor que adapte con winget o scoop. En Linux, apt o dnf.
¿Necesito Anthropic API Key?
Si usás Claude Code como IDE, no. Usa tu sesión de login. Sólo la necesitás si hacés llamadas directas a la API desde código.
¿Puedo sumar mis propios agentes?
Sí. Corré ./tooling/new-agent.sh — wizard interactivo. Es extensible por diseño.
¿Qué pasa si se rompe algo a la mitad?
El agente rollback-manager vuelve al último commit verde. El doctor.sh diagnostica qué está mal.
¿Puedo trabajar en equipo?
Sí. Invitás colaboradores a tu repo de GitHub. Cada uno clona, corre install, y tiene los mismos 32 agentes. El guardián audita antes de cada merge.
