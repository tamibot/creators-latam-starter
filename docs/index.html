PASO 01 · INSTALAR

Cloná y esperá.

Un único comando. El instalador detecta qué te falta, descarga solo lo necesario, y deja listo el entorno con los 32 agentes.

🍿 Puede tardar 15–30 min la primera vez. Son descargas de tools + modelos de IA (~2GB). Dale todos los permisos cuando pregunte — es seguro y 100% auditable.
~/mi-proyecto — install.sh
$ git clone creators-latam/starter mi-proyecto
Cloning into 'mi-proyecto'... done.
$ cd mi-proyecto && ./tooling/install.sh

PASO 02 · ENTREGABLE

Definí qué entregás.

Antes de tocar código, el agente de onboarding te guía en un loop iterativo hasta que quede 100% claro qué recibe el cliente.

PRO TIP Mandá una nota de voz de 20 minutos detallando todo: tipo de cliente, qué espera ver, cómo lo usa. Gemini la entiende perfecto y la convierte en especificación.
💡 El loop se cierra solo cuando sabés dónde se guarda, qué columnas tiene, con qué frecuencia se genera y cómo lo firma el cliente. Ni antes.

PASO 03 · STACK

Validá el stack.

PREGUNTAS FRECUENTES

Lo que más preguntan.

No. Los 5 pasos están pensados para que cualquiera que clone y siga el onboarding pueda lanzar un proyecto. El sistema pregunta, vos respondés (incluso por nota de voz).
Primera vez en Mac vacía: 15–30 min. Siguientes corridas: segundos — el script detecta qué ya tenés y salta lo instalado.
El script asume macOS. En Windows pedile a Claude/Cursor que adapte con winget o scoop. En Linux, apt o dnf.
Si usás Claude Code como IDE, no. Usa tu sesión de login. Solo la necesitás si hacés llamadas directas a la API desde código.
Sí. Corré ./tooling/new-agent.sh — wizard interactivo que valida nombre único, frontmatter correcto, y te abre el archivo listo para editar.
El agente rollback-manager vuelve al último commit verde (identificado por CI). El script ./tooling/doctor.sh diagnostica issues con modo --fix.
Sí. Invitás colaboradores a tu repo de GitHub. Cada uno clona, corre install, y tiene los mismos 32 agentes. El guardian-reglas audita antes de cada merge. El onboarder-colaborador genera welcome personalizado.
